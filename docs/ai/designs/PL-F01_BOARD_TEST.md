本文给出 **流水版** 全套验收材料：要写的 **指令字 / 寄存器 / 内存初值**、**运行后预期结果**，以及按 **流水机器周期**（非顺序版 W1/W2/W3）列出的 **逐步控制字**。面板操作与 SW 模式见 [HW-F01_BOARD_TEST](./HW-F01_BOARD_TEST.md)。

## 1. 与顺序版的观测差异

| 顺序版（`hardwired_ctrl`） | 流水版（`hardwired_ctrl_pipe`） |
|---------------------------|--------------------------------|
| W1 只亮 `LIR+PCINC` | **短指令** EX 型拍可同时亮 **执行字 + `LIR+PCINC`** |
| W2 只亮执行字 | **LD/ST**：EX 与 MEM 型拍 **均不取指**（保 IR3:0） |
| W3 为 LD/ST 第三拍 | MEM 型拍对应原 W3；MEM 后单独一拍 IF 下一条 |

**本文「周期」定义：** SW=000 连续运行时，时序发生器每推进一轮全局节拍，你在控制字指示灯上观察到的一帧组合输出，记为 **cyN**。流水状态在 **T3↓** 更新；cyN 表的是该拍稳定期间的 **组合输出**。

**说明：** 流水 v1 RUN 模式 **不拉 `SHORT`**；`LONG` 仅在 LD/ST 的 **EX 型拍**有效。`LIAR`/`IABUS` 在 `top.v` 中恒为 0。

## 2. 指令编码（HD7 = MSB）

权威字段定义见 [REQUIREMENTS_ANALYSIS §3.2](./REQUIREMENTS_ANALYSIS.md#32-指令级需求图片-45)：

```
HD7 HD6 HD5 HD4 | HD3 HD2 | HD1 HD0
   操作码 4 位   |   Rd    |   Rs
```

寄存器：**R0=00，R1=01，R2=10，R3=11**。

| 格式 | 低 4 位含义 |
|------|-------------|
| `ADD/SUB/AND Rd,Rs` | `IR3:2`=目标 **Rd**，`IR1:0`=源 **Rs** |
| `INC Rd` | `IR3:2`=**Rd**（`IR1:0` 可写 00） |
| `LD Rd,[Rs]` | `IR3:2`=目标 **Rd**，`IR1:0`=地址寄存器 **Rs** |
| `ST Rs,[Rd]` | **`IR1:0`=被写的源寄存器 Rs**，**`IR3:2`=地址寄存器 Rd**（与 LD 助记符顺序不同，位域规则相同） |
| `JC/JZ imm` | `IR3:0`=4 位相对偏移 |
| `JMP Rd` | `IR3:2`=**Rd**（PC←Rd） |

**拼码公式（二进制）：** `{opcode[3:0], field_high[1:0], field_low[1:0]}`

### 2.1 本测试用到的全部指令字（已校对低 4 位）

| 指令 | 机器码 | HD7…HD0 | 低 4 位解读 |
|------|--------|---------|-------------|
| `ADD R0,R1` | **0x11** | `0001_0001` | Rd=R0, Rs=R1 |
| `SUB R2,R0` | **0x28** | `0010_1000` | Rd=R2, Rs=R0 |
| `SUB R0,R0` | **0x20** | `0010_0000` | Rd=R0, Rs=R0 |
| `AND R0,R1` | **0x31** | `0011_0001` | Rd=R0, Rs=R1 |
| `INC R0` | **0x40** | `0100_0000` | Rd=R0 |
| `INC R3` | **0x4C** | `0100_1100` | Rd=R3 |
| `LD R1,[R2]` | **0x56** | `0101_0110` | Rd=R1, Rs=R2 |
| `ST R0,[R2]` | **0x68** | `0110_1000` | 源 Rs=R0, 地址 Rd=R2 |
| `JC +1` | **0x71** | `0111_0001` | offset=+1 |
| `JZ +2` | **0x82** | `1000_0010` | offset=+2 |
| `JMP R2` | **0x98** | `1001_1000` | Rd=R2 |
| `STP` | **0xE0** | `1110_0000` | — |

> 控制器译码 **只看 IR7:4**；**数据写哪寄存器 / 哪段内存** 由低 4 位决定，必须写对。
>
> **常见笔误（勿用）：** `0x5A`=`LD R2,[R2]`；`0x51`=`LD R0,[R1]`（会进 **R0**）；`0x62`=`ST R2,[R0]`（不是 `ST R0,[R2]`）。

### 2.2 控制字缩写

| 标记 | 含义 |
|------|------|
| **IF** | `LIR` + `PCINC` |
| **EX-ADD** | `ABUS` `DRW` `CIN` `S3` `S0` `LDZ` `LDC`（`M`=0） |
| **EX-SUB** | `ABUS` `DRW` `S2` `S1` `LDZ` `LDC`（`M`=0, `CIN`=0） |
| **EX-AND** | `ABUS` `DRW` `M` `S3` `S1` `S0` `LDZ` |
| **EX-INC** | `ABUS` `DRW` `LDZ` `LDC`（`M`=0, `CIN`=0） |
| **EX-LD** | `ABUS` `LAR` `LONG` `M` `S3` `S1`（`S2`=0, `S0`=0） |
| **EX-ST** | `ABUS` `LAR` `LONG` `M` `S3` `S2` `S1` `S0` |
| **MEM-LD** | `MBUS` `DRW` |
| **MEM-ST** | `ABUS` `MEMW` `M` `S3` `S1`（`S2`=0, `S0`=0） |
| **EX-JC** | `PCADD`（仅当 **C=1**；本测试 C=0 → 无输出） |
| **EX-JZ** | `PCADD`（仅当 **Z=1**） |
| **EX-JMP** | `ABUS` `LPC` `M` `S3` `S2` `S1` `S0` |
| **EX-STP** | `STOP` |
| **—** | 该拍无上述有效输出（复位后首拍 IF 时 EX 列为 —） |

**EX+IF** = EX 列与 **IF** 同时亮。**MEM** 拍 **不含 IF**（bubble）。

## 3. 测试程序 A — INC 循环 + JMP（你已测过的类型）

### 3.1 程序

| 地址 | 机器码 | 指令 |
|------|--------|------|
| 0x00 | 0x40 | `INC R0` |
| 0x01 | 0x98 | `JMP R2` |

### 3.2 初值

| 项目 | 值 | 灌入 |
|------|-----|------|
| R0 | 0x00 | SW=100，第 1 组 W1 写 `0000_0000` |
| R2 | **0x00** | SW=100，第 3 组 W1 写 `0000_0000`（JMP 目标 = 0x00） |
| 0x00 | 0x40 | SW=001 写存 |
| 0x01 | 0x98 | SW=001 写存 |

R1、R3 任意；本程序不用。

### 3.3 预期结果

| 检查项 | 预期 |
|--------|------|
| 连续运行 | R0 **不断 +1**；PC 在 0x00↔0x01 间循环 |
| 控制字 | 稳态每拍 **EX-INC + IF** 同亮；`JMP` 拍 **EX-JMP** 亮、**无 IF** |

### 3.4 逐步控制字（CLR# 后 PC=0，稳态前两拍 + 跳转拍）

| 周期 | PC→ | IR 中指令 | 型 | 有效控制字 | 说明 |
|------|-----|-----------|-----|------------|------|
| cy1 | 0→1 | 取入 INC | EX | **IF** | 首拍 `instr_cached=0`，只取指 |
| cy2 | 1→2 | INC + 取入 JMP | EX | **EX-INC** + **IF** | 重叠：R0+1 同时取下一条 |
| cy3 | 2→0 | JMP + 取入 INC | EX | **EX-JMP** | `LPC`；**无 IF**（分支 flush） |
| cy4 | 0→1 | INC + … | EX | **EX-INC** + **IF** | 回到循环稳态 |

## 4. 测试程序 B — 全指令综合（推荐主验收）

覆盖 **ADD / SUB / AND / INC / LD / ST / JZ / JC / STP**；**JMP** 在程序 C 单独测。

### 4.1 程序（地址 0x00–0x0C）

| 地址 | 机器码 | 指令 | 作用 |
|------|--------|------|------|
| 0x00 | 0x11 | `ADD R0,R1` | R0 ← 5+3 |
| 0x01 | 0x56 | `LD R1,[R2]` | R1 ← M[0x10] |
| 0x02 | 0x68 | `ST R0,[R2]` | M[0x10] ← R0 |
| 0x03 | 0x31 | `AND R0,R1` | R0 ← R0 & R1 |
| 0x04 | 0x4C | `INC R3` | R3 ← R3+1 |
| 0x05 | 0x28 | `SUB R2,R0` | R2 ← R2−R0 |
| 0x06 | 0x20 | `SUB R0,R0` | R0 ← 0，**Z=1**，**C=1**（0−0 无借位） |
| 0x07 | 0x82 | `JZ +2` | Z=1 → 跳过 0x08、0x09，PC→0x0A |
| 0x08 | 0x4C | `INC R3` | （JZ 跳过） |
| 0x09 | 0x4C | `INC R3` | （JZ 跳过） |
| 0x0A | 0x71 | `JC +1` | **C=1 → 跳过 0x0B**，PC→0x0C |
| 0x0B | 0x4C | `INC R3` | （JC 跳过） |
| 0x0C | 0xE0 | `STP` | 停机 |

### 4.2 运行前初值（全部要写）

#### 寄存器（SW = **100**，每组 W1/W2 各单步一次）

| 顺序 | 拍 | HD7…HD0 | 写入 |
|------|-----|---------|------|
| R0 | W1 | `0000_0101` | **0x05** |
| R0 | W2 | （无关） | 单步 |
| R1 | W1 | `0000_0011` | **0x03** |
| R1 | W2 | （无关） | 单步 |
| R2 | W1 | `0001_0000` | **0x10** |
| R2 | W2 | （无关） | 单步 |
| R3 | W1 | `0000_0000` | **0x00** |
| R3 | W2 | （无关） | 单步 |

#### 内存 M[0x10]（SW = **001**，地址+数据两拍）

| 拍 | HD7…HD0 | 含义 |
|----|---------|------|
| 1 | `0001_0000` | 地址 **0x10** |
| 2 | `1010_1010` | 数据 **0xAA** |

#### 程序字节（SW = **001**，逐地址写入）

| 地址 HD | 数据 HD | 写入 |
|---------|---------|------|
| `0000_0000` | `0001_0001` | 0x00 ← **0x11** |
| `0000_0001` | `0101_0110` | 0x01 ← **0x56** |
| `0000_0010` | `0110_1000` | 0x02 ← **0x68** |
| `0000_0011` | `0011_0001` | 0x03 ← **0x31** |
| `0000_0100` | `0100_1100` | 0x04 ← **0x4C** |
| `0000_0101` | `0010_1000` | 0x05 ← **0x28** |
| `0000_0110` | `0010_0000` | 0x06 ← **0x20** |
| `0000_0111` | `1000_0010` | 0x07 ← **0x82** |
| `0000_1000` | `0100_1100` | 0x08 ← **0x4C** |
| `0000_1001` | `0100_1100` | 0x09 ← **0x4C** |
| `0000_1010` | `0111_0001` | 0x0A ← **0x71** |
| `0000_1011` | `0100_1100` | 0x0B ← **0x4C** |
| `0000_1100` | `1110_0000` | 0x0C ← **0xE0** |

### 4.3 操作步骤

| 步骤 | SW | 操作 |
|------|-----|------|
| B0 | — | **CLR#** |
| B1 | 100 | 写 R0–R3（§4.2） |
| B2 | 001 | 写 M[0x10]=0xAA；写程序 0x00–0x0C |
| B3 | 010 | 可选：读 M[0x10] 应为 **AA** |
| B4 | 000 | **CLR#**（PC←0）→ 单步或连续至 **STP** |
| B5 | 011 | 读 R0–R3 |
| B6 | 010 | 读 M[0x10] |

### 4.4 预期最终结果

| 检查项 | 预期 |
|--------|------|
| R0 | **0x00** |
| R1 | **0xAA** |
| R2 | **0x08**（0x10−0x08） |
| R3 | **0x01**（仅 0x04 一次 `INC R3`；JZ 跳过 0x08/0x09；**JC 跳过 0x0B**） |
| M[0x10] | **0x08**（ST 写入） |
| 停机 | **STOP** 在 STP 的 EX 拍有效 |

### 4.5 逐步控制字（程序 B，访存排他 IF）

| 周期 | PC | cache | 型 | 有效控制字 | 备注 |
|------|-----|-------|-----|------------|------|
| cy1 | 0x00 | — | EX | **IF** | 取 `ADD` |
| cy2 | 0x01 | ADD | EX | **EX-ADD** + **IF** | R0←8；取 `LD` |
| cy3 | 0x02 | LD | EX | **EX-LD** | **无 IF**；IR 保持 LD |
| cy4 | 0x02 | LD | MEM | **MEM-LD** | R1←M[R2]；**无 IF** |
| cy5 | 0x03 | — | EX | **IF** | 取 `ST` |
| cy6 | 0x04 | ST | EX | **EX-ST** | **无 IF** |
| cy7 | 0x04 | ST | MEM | **MEM-ST** | M[R2]←R0 |
| cy8 | 0x05 | — | EX | **IF** | 取 `AND` |
| cy9 | 0x06 | AND | EX | **EX-AND** + **IF** | 取 `INC R3` |
| cy10 | 0x07 | INC | EX | **EX-INC** + **IF** | R3←1 |
| cy11 | 0x08 | SUB | EX | **EX-SUB** + **IF** | R2←8 |
| cy12 | 0x09 | SUB | EX | **EX-SUB** + **IF** | R0←0；**Z=1，C=1** |
| cy13 | 0x0A | JZ | EX | **EX-JZ** | 无 IF；PC→0x0A |
| cy14 | 0x0B | — | EX | **IF** | 取 `JC` |
| cy15 | 0x0C | JC | EX | **EX-JC** | **C=1**：`PCADD`；无 IF；PC→0x0C |
| cy16 | 0x0D | — | EX | **IF** | 取 `STP` |
| cy17 | 0x0E | STP | EX | **EX-STP** + **IF** | **STOP** |

**流水特征核对：**

1. **cy2、cy9–cy11**：短指令 **EX+IF** 同亮。
2. **cy3–cy4、cy6–cy7**：访存指令 **只 EX / 只 MEM**，**`LIR`/`PCINC` 不亮**。
3. **cy5、cy8、cy14、cy16**：LD/ST 或分支后的 **纯 IF** 拍。
4. **cy13、cy15**：分支 taken，仅 **`PCADD`**（无 `LIR+PCINC`）。
5. **R3=0x01**：`SUB R0,R0` 后 **C=1**，`JC +1` 跳过 0x0B 的 `INC R3`。

> 访存排他后流水版约 **17** 个机器周期（顺序版约 28 W 拍）。

## 7. 单拍控制字明细（便于对灯）

以下为 **EX 型拍** 各指令 **执行侧** 完整信号（**不含 IF**）。MEM 型拍见 §2.2。

| 指令 | 点亮的输出（=1） |
|------|------------------|
| ADD | `ABUS` `DRW` `CIN` `S0` `S3` `LDZ` `LDC` |
| SUB | `ABUS` `DRW` `S1` `S2` `LDZ` `LDC` |
| AND | `ABUS` `DRW` `M` `S0` `S1` `S2` `S3` `LDZ` |
| INC | `ABUS` `DRW` `LDZ` `LDC` |
| LD（EX） | `ABUS` `LAR` `LONG` `M` `S1` `S3` |
| LD（MEM） | `MBUS` `DRW` |
| ST（EX） | `ABUS` `LAR` `LONG` `M` `S0` `S1` `S2` `S3` |
| ST（MEM） | `ABUS` `MEMW` `M` `S1` `S3` |
| JC（taken） | `PCADD` |
| JZ（taken） | `PCADD` |
| JMP | `ABUS` `LPC` `M` `S0` `S1` `S2` `S3` |
| STP | `STOP` |
| 取指 | `LIR` `PCINC` |

**ALU 辅助：** `M`=1 为逻辑运算；`CIN` 仅 ADD 为 1。`S3 S2 S1 S0` 按上表。

## 8. 常见问题

| 现象 | 可能原因 |
|------|----------|
| 看不到 EX+IF 同亮 | 仍在顺序 bit；或首拍 / MEM 拍（正常无重叠） |
| LD 后结果错 | M[地址] 未写；R2 初值不是 0x10 |
| JZ 没跳过 | `SUB R0,R0` 未执行，Z 仍为 0 |
| R3 是 01 不是 02 | **正确**：`SUB R0,R0` 置 **Z=1 且 C=1**；`JC +1` 跳过 0x0B |
| LD 写错寄存器 | 检查指令字低 4 位（`LD R1,[R2]`=**0x56**） |
| STO 手动模式异常 | **T3→C10** 飞线未接 |
