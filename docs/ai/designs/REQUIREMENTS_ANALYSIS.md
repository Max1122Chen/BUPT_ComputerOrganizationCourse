## TL;DR

在 TEC-PLUS **固定数据通路**上实现 **硬布线控制器**（Verilog）。**基础** = 顺序控制，覆盖课设基本指令集与手动调试通路；**进阶** = 三级流水重叠 + 数据/控制冒险处理 + 性能量化。本文件为审批用总需求；实施按 Feature/Slice 执行，**审批前不写 RTL**。

## Scope

- **In:** 控制器设计、仿真、ISE 综合、上板、调试日志、答辩材料依据
- **Out:** 数据通路改动、微程序控制器、拓展层次完整中断现场（IRET 仅实现 ISA 级控制字，不做拓展专题）、题目 B

## 2) 平台与接口需求

### 2.1 硬件平台

| 项 | 规格 |
|----|------|
| 实验箱 | TEC-PLUS |
| FPGA | Xilinx Spartan-6 XC6SLX9-2FTG256 |
| 控制器模式 | 拨码至 **硬连线**（非微程序） |
| 工具 | ISE 14.7、IMPACT 烧录 `.bit` |

### 2.2 控制器边界（不实现的部分）

实验箱已提供（图片-42）：ALU、R0–R3、双口 RAM、PC、IR、AR、时序发生器、总线多路器等。

**本仓库只实现硬布线控制器模块** `hardwired_ctrl`（及流水扩展），输出课设规定的控制信号。

### 2.3 端口需求（与图片-47 一致）

**输入：** `CLR#`, `T3`, `SWA`, `SWB`, `SWC`, `IR4`–`IR7`, `W1`, `W2`, `W3`, `C`, `Z`

**输出：** `DRW`, `PCINC`, `LPC`, `LAR`, `PCADD`, `ARINC`, `SELCTL`, `MEMW`, `STOP`, `LIR`, `LDZ`, `LDC`, `CIN`, `S0`–`S3`, `M`, `ABUS`, `SBUS`, `MBUS`, `SHORT`, `LONG`, `SEL0`–`SEL3`

### 2.4 时序需求（图片-44）

- 时序发生器产生 `W1`→`W2`→`W3`；每个 Wi 覆盖完整 `T1`–`T3` 序列（4 个 MF 节拍）
- 控制器在 **当前 Wi 为高且 IR/标志稳定** 时输出对应微操作
- `SHORT` / `LONG` 通知时序部件本指令为 **2 周期** 或 **3 周期** 指令

## 3) 功能需求 — 基础层次（CTL-F01）

### 3.1 指令执行（SW = 000 取指运行）

当 `SWC SWB SWA = 000` 进入指令执行流程（图片-43 下半部分）：

| 阶段 | 需求 ID | 行为 |
|------|---------|------|
| W1 取指 | FR-FETCH | 断言 `LIR`, `PCINC`；`PCINC` 不得早于指令稳定锁存（防吞指令） |
| W2 执行 | FR-EX | 按 `IR7–IR4` 译码输出 ALU/访存/跳转控制（见 §3.2） |
| W3 访存 | FR-MEM | 仅 LD/ST 第二拍；其余指令无 W3 有效操作 |

### 3.2 指令级需求（图片-45）

| Opcode | 指令 | 周期 | W2 控制字（摘自流程图） | W3 |
|--------|------|------|-------------------------|-----|
| 0001 | ADD Rd,Rs | 2 | S=1001,CIN,ABUS,DRW,LDZ,LDC | — |
| 0010 | SUB Rd,Rs | 2 | S=0110,ABUS,DRW,LDZ,LDC | — |
| 0011 | AND Rd,Rs | 2 | M,S=1011,ABUS,DRW,LDZ | — |
| 0100 | INC Rd | 2 | S=0000,ABUS,DRW,LDZ,LDC | — |
| 0101 | LD Rd,[Rs] | 3 | M,S=1010,ABUS,LAR,LONG | DRW,MBUS |
| 0110 | ST Rs,[Rd] | 3 | M,S=1111,ABUS,LAR,LONG | S=1010,M,ABUS,MEMW |
| 0111 | JC | 2 | C=1 → PCADD | — |
| 1000 | JZ | 2 | Z=1 → PCADD | — |
| 1001 | JMP Rd | 2 | M,S=1111,ABUS,LPC | — |
| 1010 | OUT Rs | 2 | **见 §3.4** | — |
| 1011 | IRET | 2 | **见 §3.4** | — |
| 1100 | DI | 2 | **见 §3.4** | — |
| 1101 | EI | 2 | **见 §3.4** | — |
| 1110 | STP | 2 | STOP | — |

**寄存器字段：** `IR3–IR2` = Rd，`IR1–IR0` = Rs；JC/JZ 的 `IR3–IR0` = 4 位相对偏移。

### 3.3 手动调试通路（图片-43 上半部分）

| SWC SWB SWA | 功能 | 需求 |
|-------------|------|------|
| 000 | 取指 | 转入 §3.1 指令流 |
| 001 | 写存储器 | STO 分支：地址/setup 与 MEMW+ARINC |
| 010 | 读存储器 | STO 分支：LAR+SHORT 或 MBUS+ARINC |
| 011 | 读寄存器 | SEL* + SELCTL + STOP |
| 100 | 写寄存器 | SBUS + SEL* + DRW + STOP |

> `STO` 为板级状态位（由数据通路反馈，控制器作为输入若未引出则需在设计中确认网表连接；否则在 `HW-F01` 上板时按实验箱手册接线）。

### 3.4 流程图未列指令（OUT / IRET / DI / EI）

图片-43 未给出四条指令微操作。**基础层次仍须支持**（课设指令表含全条）。

| 指令 | 暂定策略 | 验证方式 |
|------|----------|----------|
| OUT Rs | W2：选通 Rs 到总线（`SEL*`+`SELCTL`+`ABUS` 或 `SBUS`）；与实验箱 OUT 设备连线一致 | 上板 + 对照微程序模式波形 |
| DI | W2：断言时序部件关中断相关控制（若仅为空操作则记录 ADR） | 上板中断演示 |
| EI | W2：开中断使能 | 上板 |
| IRET | W2：中断返回序列（`LPC` 或专用组合）；**非拓展专题** | 上板简化测试 |

风险：四条指令在 `CTL-F01-S08` 单独切片，**以微程序控制器黄金波形为准**做回归；偏差写 ADR。

### 3.5 基础层次非功能需求

| ID | 需求 |
|----|------|
| NF-01 | Verilog 可综合，无 latch（除刻意设计） |
| NF-02 | 控制信号命名与实验箱一致 |
| NF-03 | 仿真可检查每条指令 W1/W2/W3 向量 |
| NF-04 | 调试日志记录于 `PROGRESS_LOG.md` |

## 5) 仿真与上板需求

### 5.1 仿真（SIM-F01）

- 不要求完整 RTL 数据通路；采用 **控制向量检查器** 或 **简化行为模型**
- 输入：IR、W1–W3、C、Z、SW；检查输出控制字是否与黄金表一致
- 集成 `scripts/verify.ps1 -Stage 1`

### 5.2 上板（HW-F01）

- `constraints/tecplus.ucf` 锁定图片-47
- ISE 工程 `ise/` 生成 `top.bit`
- IMPACT 下载；拨码硬连线；运行课设/教师测试程序
- 记录现象于 `PROGRESS_LOG.md` 与可选 `bugs/`
